Product Line Profitability & Margin Performance Analysis
Nassau Candy Distributor
Project Objective
The objective of this project is to analyze product-wise profitability and margin performance for Nassau Candy Distributor in order to identify high-performing and low-performing products based on sales, gross profit, and gross margin percentage.
 Tools Used
Microsoft Excel
Pivot Tables
Calculated Fields
GitHub (Project Repository)
 Dataset
Source: Nassau Candy Distributor sales dataset
File: Nassau Candy Distributor.csv
Key Metrics Analyzed
Total Sales by Product
Total Gross Profit by Product
Weighted Gross Margin %
Product-level Profitability Comparison
 Analysis Performed
Created Gross Profit and Gross Margin % calculations
Built Pivot Tables to summarize:
Product-wise Sales
Product-wise Gross Profit
Weighted Gross Margin %
Identified top and bottom performing products based on profitability
 Key Business Insights
Certain products such as Wonka Bar variants contribute the highest gross profit
Some low-margin products generate sales but reduce overall profitability
High-margin products should be prioritized for sales and marketing focus

 Business Impact
This analysis helps management to:
Focus on high-margin products
